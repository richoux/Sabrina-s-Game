# Sabrina-s-Game
Constructive algorithm building valid solutions to Sabrina's Game.

## Compile (requires a C++20 compiler)
```shell
$> make
```

## Run
For getting one solution and display it with a pretty ASCII drawing:
```shell
$> bin/sabrinasgame WIDTH HEIGHT
```
For instance:
```shell
$> bin/sabrinasgame 4 6
```

For running it multiple times and getting a simple solution display:
```shell
$> bin/sabrinasgame WIDTH HEIGHT RUNS
```
For instance, for 10 runs:
```shell
$> bin/sabrinasgame 4 6 10
```
